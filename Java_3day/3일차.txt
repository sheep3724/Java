자바의 기본
-------------------
변수 (기초형) - 정수(int), 실수(double), 논리(boolean), 문자(char)
int a = 10;

참조형 - 변수에 주소가 저장 기초형빼고 나머지 다 배열, 컬렉션,클래스
int b = a;

연산자 - 산술, 비교, 논리, 증감

제어문 - 조건문, 반복문
	 if(조건), if(조건)~else~ switch
	 while, for, do~while

():소괄호 => 메소드(함수)
{}:중괄호 => 포함,사용영역
[]:대괄호 => 배열

변수- 이름이 다 달라야 함
      반드시 타입을 선언해주어야 함
      변수가 많아지면 다 이름도 달라야 하고, 선언도다해주어야함.
배열 - 이름을 하나로, 선언도 한번만
