대상(부품으로 만들 대상) => class
----
정적특징 - 이름, 나이, 연락처, 주소 => 변수
동적특징 => 메소드

메소드 만들때 고려해할 점.
------------------------
1. 반환값
반환값이 있는 경우 - 반환타입 : return
반환값이 없는 경우 - void

2. 입력값
입력값이 있는 경우 - 입력값을 순서대로 써줌
입력값이 없는 경우 - ()

다형성(오버로딩) - 메소드 오버로딩
---------------------------------
메소드를 만들때 같은 기능이라면
메소드 이름을 동일하게 하나로
만들어 쓸 수 있다.
"하나의 이름"으로 여러가지 형태의 메소드를 만들 수 있다.

생성자(constructor:컨스트럭터)
new로 객체가 생성되면서, 자동으로 복사된 변수에 값을 넣어주는
기능을 가진 메소드
규칙: 리턴타입X
      생성자메소드 이름은 클래스이름과 동일
      객체생성될 때 자동으로 호출

정적변수 : static
객체의 생성과는 상관없이
클래스당 하나만 가지고 있으면서
객체들간 공유하면서 사용하는 변수
